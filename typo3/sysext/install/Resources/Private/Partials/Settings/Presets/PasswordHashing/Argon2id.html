Use Argon2id if you are using PHP >= 7.3 on all instances (local, test, production, ...)!
Argon2id is a modern key derivation function. It provides better resistance to some forms of attack compared to Argon2i. Argon2id is not available on this system. If you want to use Argon2id make sure you are using PHP >= 7.3 with Argon support.
